# EcoBot
A Economy Bot made using py-cord

## Usage

Install the required modules

```
pip install -r requirements.txt
```

Create a file `data.json` with these contents

```
{
    "name" : "Bot Name",
    "token" : "Bot Token",
    "mongo" : "Mongo Url"
}
```

For mongodb url, you can get a free 500 mb cluster on mongo db.
